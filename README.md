# 논문 도우미 (Thesis Helper)

> 대학생이 논문을 더 쉽고 정확하게 작성할 수 있도록 **템플릿, 자동 인용, 형식 검사**를 제공하는 웹 기반 논문 작성 지원 플랫폼

## 무엇을 하는 프로젝트인가

논문 도우미는 대학생들이 논문 작성 과정에서 겪는 형식 설정, 각주 인용, 참고문헌 목록 작성의 어려움을 해결하기 위해 개발된 웹 서비스입니다. 복잡한 학술 형식을 자동으로 처리하여 연구자가 본문의 내용에 더 집중할 수 있도록 돕습니다.

**핵심 기능:**
*   **논문 작성 가이드**: 초록부터 참고문헌까지 각 섹션별 작성법을 안내합니다.
*   **자동 각주/참고문헌 생성**: 자료 정보 입력 시 4가지 주요 형식(한국연구재단, APA, MLA, Chicago)으로 자동 변환합니다.
*   **표준 템플릿 제공**: 학술지 및 학위 논문 기준에 맞춘 .docx 파일을 즉시 다운로드할 수 있습니다.
*   **데이터 로컬 저장**: 입력한 자료 목록은 브라우저에 안전하게 저장되어 재방문 시에도 유지됩니다.

## 사용 방법

본 프로젝트는 Next.js 환경에서 구동됩니다. 아래 명령어를 통해 로컬에서 실행할 수 있습니다.

```bash
# 1. 프로젝트 클론
git clone <repository-url>
cd thesis-helper

# 2. 의존성 설치
npm install

# 3. 개발 서버 시작
npm run dev

# 4. 브라우저 접속
# http://localhost:3000
```

## 사용 예시 ★ 실제 출력/화면

### 1. 논문 작성 가이드 (`/guide`)
사이드바를 통해 논문 구성 요소를 탐색하고, 각 단계별 체크리스트와 작성 팁을 확인할 수 있습니다.

### 2. 각주 및 참고문헌 생성 (`/citation`)
*   **입력**: 단행본, 학술논문, 인터넷 기사 중 유형을 선택하고 정보를 입력합니다.
*   **변환**: 상단 탭에서 형식을 선택하면 즉시 해당 스타일로 참고문헌 목록이 정렬 및 변환됩니다.
*   **결과**: 생성된 텍스트를 복사하거나 파일로 다운로드하여 논문에 바로 적용합니다.

### 3. 논문 템플릿 다운로드 (`/templates`)
용도에 맞는 카드(학술논문, 졸업논문, 보고서)를 선택하여 설정된 서식이 적용된 워드 파일을 다운로드합니다.

## 입력 / 출력 (표)

| 구분 | 입력 항목 | 출력 형식 (예시) |
| :--- | :--- | :--- |
| **단행본** | 저자, 제목, 출판사, 발행연도 | 홍길동, 『논문 작성법』, 서울: ○○출판사, 2024. |
| **학술논문** | 저자, 제목, 학술지명, 발행연도 | 이영희, "연구 방법론", 『학술지』, 2023. |
| **인터넷 기사** | 저자, 제목, 출처, 발행연도 | 박민준, "최신 기술 동향", 뉴스매체, 2024. |

## 아직 안 되는 것 ★ 부숴보며 찾은 것

*   **HWP 파일 생성**: 현재는 .docx 파일만 지원하며, 한글(HWP) 파일 다운로드는 준비 중입니다.
*   **서버 동기화**: 데이터가 브라우저(localStorage)에만 저장되므로, 기기 간 데이터 동기화는 지원되지 않습니다.
*   **자동 형식 검사**: 작성된 논문 파일을 업로드하여 형식을 검증하는 기능은 추후 업데이트 예정입니다.
*   **AI 교정**: 문장 교정 및 논리 구조 체크 기능은 현재 범위에 포함되지 않았습니다.
